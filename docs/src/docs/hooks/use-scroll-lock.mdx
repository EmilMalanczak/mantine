## Usage

Use this hook when you want to prevent user from scrolling on document level,
for example, with modals and drawers.

Example modal component, whenever modal is opened, scroll is locked at current scroll position.
After lock it adds padding to `body` equal to size of removed scrollbar and saves it ass `css-variable` as `--removed-scroll-width` which is removed after unlock. 

To prevent move on layout components like `Navbar` component you can add `padding-right: var(--removed-scroll-width, 0px))`.

Hook accepts two arguments:

 – optional boolean value that controls if scroll is locked

 – optional options object:
 - `disableBodyPadding` optional boolean that controls if `--removed-scroll-width` should be handled, default to `false`

Hooks returns an array of same usage as `useState` hook.
```tsx
import { useScrollLock } from '@mantine/hooks';

function Modal({ opened }) {
  useScrollLock(opened);

  if (!opened) {
    return null;
  }

  return <div>My modal</div>;
}
```
